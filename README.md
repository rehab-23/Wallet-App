# Wallet-App

### Info
Bei diesem Projekt handelt es sich um eine Banking-Simulation/Wallet-App.  
Dieses Projekt wurde als Portfolioprojekt erstellt und ist voll funktionsfähig.  
Die Entwicklung und das Testing erfolgten unter Windows 11 mit XAMPP als lokaler Serverumgebung.  

### Projekt in Betrieb nehmen
1. [XAMPP Downloaden unter:](https://www.apachefriends.org/de/download.html)
1. Projekt-Fork erstellen
1. Fork auf XAMPP-Server in den Ordner htdocs legen.  
Auf einem Windows-PC befindet sich der Ordner htdocs standardmäßig unter folgendem Pfad:  C:\xampp\htdocs\HIER-DEIN-PROJEKT-ABLEGEN
1. XAMPP Apache + mysql starten
1. Datenbank erstellen: SQl-Script 'db_create.sql' aus den Projekt-Dateien in der XAMPP-Oberfläche importieren
1. [Projekt im Webbrowser mittels URL aufrufen:](http://localhost/Wallet-App/index.php)
